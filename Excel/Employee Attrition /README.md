# Employee Attrition & Workforce Risk Analysis (Excel)

## Overview
Excel-based analysis to identify employee attrition drivers, high-risk segments, and early exit patterns using HR data.

---

## Dataset
- Source: **IBM HR Analytics Attrition Dataset (Kaggle)**
- Records: 1,470 employees
- Data includes demographics, job roles, tenure, satisfaction, overtime, and attrition status

---

## Key Metrics
- **Total Employees:** 1,470  
- **Employees Left:** 237  
- **Overall Attrition Rate:** 16.12%  
- **High-Risk Employees Identified:** 56  
- **Average Tenure (All Employees):** 7.01 years  
- **Average Tenure (Leavers):** 5.13 years  
- **Early Attrition Rate:** 24.88%
